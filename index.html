C++ Computer Game : Reversi /Othello

Successfully developed a computer game utilizing C++ in Code Blocks. The Game consisted of two algorithms that played against each other in a game of Othello/ Reversi. The size of the game board was variable and was the main input of the user in a .txt file. Upon the random moves of each of the algorithms, a record was kept in order to give a play-by-play summary of the results at the end. This was then sent to a .txt file. The Algorithm that manages to capture the most pieces on the board, wins.
